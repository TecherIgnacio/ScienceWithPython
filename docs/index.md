## ¿De qué trata este curso?

Bienvenido/a a **Ciencia con Python**, un curso donde vas a aprender a programar
mientras resolvés preguntas científicas reales:

- ¿Cuánto tarda la luz en llegar del Sol a la Tierra?
- ¿Cómo calcular la trayectoria de un cohete?
- ¿Qué pasa con la temperatura de un objeto en el espacio?

No necesitás saber nada de programación para empezar. ¡Vamos desde cero!

</div>
<div class="lang-en" markdown="1">

## What is this course about?

Welcome to **Science with Python**, a course where you will learn to code
while solving real scientific questions:

- How long does it take light to travel from the Sun to Earth?
- How do you calculate a rocket's trajectory?
- What happens to the temperature of an object in space?

No programming knowledge required. We start from scratch!

</div>
